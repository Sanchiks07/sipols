git komandas

cd - pārvietoties


ls - apskatīt saturu


touch - izveidot failu


git init - uztaisīt par git mapi


gti branch -M main - pārtaisa citu galveno brach


git add. - pievienot visu ko jau esi izdarījis


git commit -m - uztaisa ziņojumu


git remote add origin "saite" - tikai pirmo reizi, lai pievienotu saiti uz savu git lapu


git push - no otrās reizes, lai viesu ievietotu (git push -u - pirmo reizi)
